# Trabajo Práctico: Aplicación con una API asignada

Hola Santiago Mateu.

Vas a crear una aplicación en Next.js y Tailwind CSS usando la API que te tocó asignada.

La idea es que armes una web prolija, clara y ordenada, donde se vea que entendiste cómo trabajar con datos reales desde una API.

## Lo que tiene que incluir

- Crear un archivo de ambiente para guardar los datos que necesites usar.
- Mostrar la información principal dentro de una card.
- Usar HTML semántico en la estructura de la página, con partes como Navbar, Main, Footer y las que hagan falta.
- Separar la app en componentes reutilizables.
- Agregar paginación. Si la API ya la trae, usarla. Si no la trae, crearla vos.
- Usar más de un endpoint cuando la API lo permita.
- Crear una página con tus datos: apellido, nombre, asignatura y mail.
- En esa misma página, explicar de manera simple qué información brinda la API que estás usando.
- Crear una página de comentarios donde se puedan guardar comentarios en un archivo JSON.
- Cada comentario tiene que guardar fecha y hora de creación, título y detalle.
- Los comentarios también tienen que mostrarse en pantalla.

## Qué se espera

- Que la aplicación funcione bien.
- Que esté ordenada y sea fácil de navegar.
- Que se note el uso de componentes.
- Que la información que se muestra tenga sentido con la API asignada.
- Que el diseño sea limpio y prolijo.
